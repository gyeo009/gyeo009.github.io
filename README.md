# gyeo009.github.io

Personal blog for writing about backend engineering, AI systems, and lessons from work.

## Stack

- Hugo
- Custom layouts and styles
- GitHub Pages

## Local writing workflow

1. Install Hugo Extended on your machine.
2. Run `hugo server -D` in the repository root.
3. Write posts in `content/posts/`.
4. Use `archetypes/default.md` as the default Markdown template.

## Notes

- The site is designed to stay text-first, readable, and easy to maintain.
- Writing on the site is intended to share generalized lessons from work without exposing sensitive company details.
