1년 2개월 정도 정규직으로 일하며 백엔드 시스템을 만들고 운영해왔습니다.

- 약 3개월 동안 Database 접근제어 Proxy를 개발했습니다.
- 약 7개월 동안 Kotlin 기반 MCP server를 개발, 관리, 운영했습니다.
- 현재는 TypeScript, Node.js, NestJS 기반 LLM chat middleware 서버를 개발하고 있습니다.

이 블로그는 포트폴리오이면서도, 실무 중 생긴 생각을 정리하는 기록장입니다.
특정 회사의 민감한 정보는 덜어내고, 대신 다음과 같은 것들을 남기려고 합니다.

- 문제를 어떻게 바라봤는지
- 어떤 설계 판단이 오래 남는지
- 운영 가능한 시스템을 만들기 위해 무엇을 신경 써야 하는지
- AI 애플리케이션 뒤편의 백엔드가 어떤 식으로 복잡해지는지

관심사는 대체로 아래에 가깝습니다.

- Backend architecture
- Database and access control
- AI systems and middleware
- Reliability, observability, and operations
- Writing down lessons learned at work

문의가 필요하면 GitHub나 메일로 연락해주세요.
